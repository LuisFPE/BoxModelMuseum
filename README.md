# BMM: Box Model Museum

## General Information

This project represents the website for a museum called the BMM Box Model Museum. This museum focuses primarily on box models and how they can be used to create works of art using geometric shapes and colors. The main structure consists of three pages: "Home," "Exhibitions," and "Tickets."

## Main Pages

- **Home:** This is the website's homepage, displaying useful information such as who we are and what we do.

- **Exhibitions (First Edition):** This is the museum's first gallery. On this page, you can view the different works of art made with box models.

- **Exhibitions (Second Edition):** This is the museum's second gallery. On this page, you can view the different works of art made with grid layout.

- **Tickets:** On this page, you can purchase tickets to the museum. Information related to prices and availability is also displayed.

## Additional Notes

To correctly view the exhibitions page, it is necessary to run the project folder from Visual Studio using the Life Server extension. This is because certain resources are hosted locally and we do not yet have an HTTPS certificate, which may cause problems with CORS policies.
